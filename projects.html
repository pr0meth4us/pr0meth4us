Cambodia License Plate Recognition System

Developed a specialized license plate recognition system for Cambodian vehicles. Fine-tuned a YOLO-based segmentation and detection model to accurately locate license plates then segment the number and province regions. Integrated with OCR (PARSeq) for text recognition. Backend built with Flask, frontend with Next.js, deployed using Docker. Focused on optimizing detection precision and inference speed for real-time usage.

Python Flask YOLO PARSeq OCR Next.js Docker

Chat Message Analyzer

A chat analysis tool that transforms exported chat logs into insights about communication patterns and relationship dynamics. Supports Telegram, Facebook, Instagram, iMessage, Discord, and TikTok exports in HTML/JSON formats. Features detailed parsing of message structure, Khmer date support, and analysis of response patterns, ghost periods, language usage, and relationship balance. Backend is built with Flask, supports CORS and file uploads, and serves a web API. (The server is not properly working yet, it's being troubleshot. Local works just fine.)
